Using XForms output formatting

Ubiquity XForms supports an extension to standard XForms mark-up, which allows the output to be formatted. The format attribute allows prefixes and suffixes to be added to the output, as well as formatting floating-point numbers.

For each of the examples we have below, there is both the markup and a live version of the code; so move your mouse over the various controls to see how the markup behaves.

Prefixes

A simple prefix can be specified as any text string, up to a '#' character: 

<xf:output value="100" format="$#">
	<xf:label>Payment:</xf:label>
</xf:output>

<xf:output value="100" format="CAN #">
	<xf:label>Payment:</xf:label>
</xf:output>

Suffixes

A suffix can be specified in the same way, but following a '#' character: 

<xf:output value="16" format="#¢">
	<xf:label>Payment:</xf:label>
</xf:output>

<xf:output value="25" format="# DEM">
	<xf:label>Payment:</xf:label>
</xf:output>

Using both prefixes and suffixes

And of course both prefixes and suffixes can be used in the same expression: 

<xf:output value="23.16" format="$#¢">
	<xf:label>Payment:</xf:label>
</xf:output>

<xf:output value="25.89" format="£#p">
	<xf:label>Payment:</xf:label>
</xf:output>

Formatting numbers

A number can have its decimal places formatted by using a decimal point in the format string, followed by a number of '#' characters to indicate the number of decimal places to apply. Note how rounding and padding with zeros takes place: 

<xf:output value="23.1534" format="#.##">
	<xf:label>Payment:</xf:label>
</xf:output>

<xf:output value="23.1534" format="#.#">
	<xf:label>Payment:</xf:label>
</xf:output>

<xf:output value="23.1534" format="#.######">
	<xf:label>Payment:</xf:label>
</xf:output>

<xf:output value="23.1534" format="#.">
	<xf:label>Payment:</xf:label>
</xf:output>

Number formatting can also be used with prefixes and suffixes: 

<xf:output value="23.4567" format="$#.##¢">
	<xf:label>Payment:</xf:label>
</xf:output>

	<xf:label>Payment:</xf:label>
