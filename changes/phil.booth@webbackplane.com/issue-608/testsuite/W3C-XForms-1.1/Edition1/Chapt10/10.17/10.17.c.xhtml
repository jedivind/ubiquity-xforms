10.17.c conditional execution of XForms actions - Automatic Focus Advancement example This test case shifts the focus to different input controls depending on if conditions have been met. If you enter three characters into the Area Code input the focus must move to the Exchange input. If you enter three characters into the Exchange input the focus must move to the Local input.
