Acura 2005 blue 11.1.l indent attribute of submission element (non-normative) Each submit control below replaces the page with the form instance data. The Submit Without indent submits the form with a submission element that has an indent attribute set to "no". The Submit With indent submits the form with a submission element that has an indent attribute set to "yes". The differences between the two submissions, if any, are dependent on your renderer.
