9.2.3.a events dispatched by toggle element 9.2.3.a events dispatched by toggle element If you are in the "in" case and you activate the Show Out Case trigger you must see an xforms-deselect(in) message followed by an xforms-select(out) message. If you are in the "out" case and you activate the Show In Case trigger you must see an xforms-deselect(out) message followed by an xforms-select(in) message. You are now in the "in" case Show Out Case xforms-select(in) xforms-deselect(in) You are now in the "out" case Show In Case xforms-select(out) xforms-deselect(out)
