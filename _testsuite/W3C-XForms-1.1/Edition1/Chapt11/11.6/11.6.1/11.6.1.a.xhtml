1 2 submit.txt 11.6.1.a resource element (non-normative) When you activate the Submit To File submit control the instance data must be submitted to the file submit.txt. If the page is replaced by the form data then you have failed the test case. After submitting the instance data activate the Load submit.txt trigger to see if the instance data was submitted to the file. The data must contain the values "1", "2", and the time that you activated the Submit To File control. This test case is non-normative because it may be difficult to ensure the form is working with the text files. Submit To File Load submit.txt
