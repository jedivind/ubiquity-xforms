10.7.a setfocus element The triggers below must shift the focus to the proper form controls when activated. The Set Focus To Shipping trigger must place the focus into the empty input control labeled Shipping. The Set Focus To First Item trigger must must place the focus into the input control containing the value "brake pad". The Set Focus To Third Item trigger must must place the focus into the input control containing the value "fan belt".
