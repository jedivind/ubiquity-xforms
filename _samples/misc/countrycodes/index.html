Country Codes

In this example we have two lists of ISO country codes, one is English and the other is French. However, both are bound to the same data field, and since the two letter ISO codes that represent the countries are the same in both languages, it doesn't matter which language is used to select a country, the internal data will always be the same.

It would also be straightforward to only show one list, but allow the user to choose which language they want the list shown in, or to determine which data to deliver, based on the user's browser settings. Some of these techniques are explored in the English/Welsh sample, available from the samples page.
